TinySAtoWWB Convertor
By Brad Baisley

--

This is a Powershell 7 script which converts CSV generated by TinySA-App into a CSV format useable in the coordination tab of Shure Wireless Workbench

Prerequisites:
Windows Powershell 7 needs to be installed

Instructions:
-Open TinytoWWBfix.ps1 in a text editor like Notepad, Notepad++ or Powershell ISE
-Edit the input file path to the CSV generated by TinySA-APP C:\YOURINPUTFILELOCATION.csv
-Edit the output file path to wherever you would like the new file to go be deposited and called  C:\YOUROUTPUTFILELOCATION.csv
-Save TinytoWWBfix.ps1 

-Open tinytowwb.bat in a text editor
-change the path to whereyour TinytoWWBfix.ps1 is located C:\PATHTOYOUR\TinytoWWBfix.ps1
-Save

-Double click tinytowwb.bat and the file you have pointed it to should be processed.
